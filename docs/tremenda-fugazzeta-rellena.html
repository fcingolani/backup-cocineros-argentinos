Tremenda fugazzeta rellena

INGREDIENTES



  • Para la salsa:

  • Tomates secos hidratados
  • Aceite de oliva
  • Ajo rallado
  • Para el relleno
  • Salchica parrillera cocida
  • Chorizo colorado
  • Queso provolone
  • Chimichurri
  • Pimienta
  • Aceite de oliva

  • Para la cobertura:

  • Cebollas rehogadas o crudas
  • Orégano
  • Pimienta

    • Hacer dos bollos de “la masa de pizza, paso a paso”, la base más alta que la cobertura. Sobre la base pintar con la salsa y cubrir con el relleno. Tapar con la masa más fina y cubrir con cebollas rehogadas, orégano, pimienta.
