Vacipan para comer al paso

INGREDIENTES



  • Para el vacipan:

  • Pan francés
  • Vacío

  • Para el aderezo de vino tinto:

  • Vino tinto
  • Ajo
  • Hierbas frescas varias
  • Ají molido
  • Sal
  • Pimienta negra
  • Agua c/n

  • Para el chimichurri clásico:

  • Aceite de girasol ½ taza
  • Vinagre de alcohol 3 cdas
  • Ajos 4
  • Perejil picado 1/2 taza
  • Orégano seco 1 cda
  • Sal gruesa 2 cas
  • Ají molido 1 cda

  • Para la salsa criolla:

  • Cebolla
  • Verdeo
  • Morrón rojo
  • Morrón verde
  • Ajo

    • PARA EL VACIPAN: Dorar el vacío entero, cortar en fetas y llevarlas a la parrilla vuelta y vuelta para que se termine de cocinar. Rociar con el aderezo de vino tinto. Llevar al pan aderezar con mitad de chimichurri mitad de salsa criolla.

    PARA EL ADEREZO DE VINO TINTO: En una botella de vino tinto incorporar las especias y todos los ingredientes dejarlo reposar de un día para el otro

    PARA EL CHIMICHURRI CLASICO: Mezclar la sal con el vinagre para que se disuelva, reservar.
    Mezclar el resto de las hiervas, agregar el aceite y luego el vinagre con la sal. Agitar bien y dejar reposar por unos días.
     
    PARA LA SALSA CRIOLLA: Cortar todo en brunoise muy chiquita, si se desea agregar aceite y vinagre
