Croquetas multicolor

INGREDIENTES



  • Para las de remolacha :

  • Remolacha rallada
  • Pure de papas seco
  • Sal
  • Pimienta
  • Cuadraditos de muzzarella

  • Apanado :

  • Galletitas de agua molidas
  • Huevos
  • Harina
  • Aceite

  • Para las de brócoli:

  • Pure de papas seco
  • Brócoli blanquedos
  • Cuadraditos de muzzarella
  • Sal
  • Pimienta negra

  • Apanado:

  • Avena
  • Huevos
  • Ajo picado
  • Perejil
  • Harina

  • Para las de zanahoria y choclo:

  • Pure de papas seco
  • Puré de zanahoria
  • Choclo rallado
  • Cuadraditos de muzzarella

  • Apanado:

  • Pan rallado
  • Huevos
  • Ajo
  • Perejil
  • Harina

  • Para freir:

  • Aceite

    • Para los 3 sabores la base es puré de papas bien seco y agregar los vegetales la idea es que quede de 3 colores bien marcados y apanados diferentes y corazón de muzzarella .


    Hacer un puré de papa con poca sal, pimienta, ajo fresco. Blanquear el brocoli con una hervida de solo 3 minutos y luego picarlo fino. Mezclar todo en un bol, retirar un puñado y formar un disco con las palmas de las manos, colocar un cubito de muzzarella y cerrar el disco formando una bolita. Pasar por huevo batido y luego empanar.

    Tener lista una sartén con aceite caliente y freir hasta obtener el dorado deseado.

    El procedimiento se repite para todas las opciones.
