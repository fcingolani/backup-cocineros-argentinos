Exquisito pastel de berenjenas y pollo gratinado

INGREDIENTES



  • Para el relleno:

  • Berenjenas asadas 200 grs
  • Pollo asado y desmenuzadas 400 grs
  • Espinacas 200 grs
  • Morrones asados 3

  • Para la masa:

  • Huevos 3
  • Queso crema 100 grs
  • Ajo 1
  • Sal
  • Pimiento negra molida
  • Leche 350 ml
  • Harina 140 grs
  • Romero

  • Para la cobertura:

  • Queso rallado c/n

    • Se intercalan las capas de berenjena, pollo, morrón y masa hasta cubrir una fuente de horno.


    Se espolvorea con queso rallado y se gratina en el horno.
