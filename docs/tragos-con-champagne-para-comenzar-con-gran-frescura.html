Tragos con champagne para comenzar con gran frescura

INGREDIENTES



  • Para la mimosa:

  • Jugo de naranja 1 parte
  • Champagne 1 parte

  • Para Rossini:

  • Pulpa o jugo de frutillas 1 parte
  • Champagne 2 partes

  • Para el Bellini:

  • Pulpa o jugo de durazno 1 parte
  • Champagne 2 partes

  • Para el Kirk Royal:

  • Licor de cassis 1 parte
  • Champagne 2 partes

  • Para el C-C Bitter Rojo y Champagne:

  • Bitter rojo 1 parte
  • Champagne 2 partes
  • Cáscara de pomelo 1 rulo

    • PARA TODOS LOS TRAGOS: En copas de champagne se debe servir primero los jugos y luego el champagne.


     

    PARA EL C-C Bitter Rojo y Champagne: Agregar el rulo de pomelo dentro de la copa.
