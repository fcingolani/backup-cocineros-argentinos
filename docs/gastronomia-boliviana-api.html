Gastronomía boliviana: Api

INGREDIENTES


  • Api
  • Harina morada 4 cdas
  • Agua 1 litro
  • Canela
  • Clavo de olor
  • Azúcar a gusto
  • Jugo de limón a gusto

    • Disolver la harina en agua fría e incorporar al agua hirviendo con canela y clavo de olor. Hervir por 3 minutos, agregar jugo de limón y azúcar.
